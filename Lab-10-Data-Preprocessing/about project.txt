Lab-10-Data-Preprocessing is a GitHub-friendly mini project that prepares your CSV dataset for machine learning. It reads data/dataset.csv using Pandas, then prints basic info (rows/columns), checks missing values using np.sum(pd.isnull(data)), and shows unique values for education (or another categorical column if needed) to understand the data categories. After that, it fills missing values using mode, removes an unnecessary id column if present, converts float columns that actually contain whole numbers into int, and prints column datatypes before/after cleaning. Finally, it splits the dataset into X (features) and y (target: loan_status), converts categorical/object columns in X into integer codes using pd.factorize(), and saves the cleaned result into outputs/cleaned_dataset.csv.

Run it with: python src/preprocessing.py (from the Lab-10 folder).
